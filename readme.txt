== Description ==

该插件支持使用阿里云存储OSS作为附件存储空间。
This is a plugin that used Aliyun Cloud Storage(Aliyun OSS) for attachments remote saving.

您可以使用其方便地将你的WP博客中的图片、附件托管在阿里云OSS服务器上，享受阿里云的高速、稳定、快捷的云存储服务。

阿里云OSS是廉价、高效的开放储存服务，您在使用OSS过后只需支付您所消耗的费用。 欲查看阿里云OSS最新收费标准请访问 阿里云OSS官方网站.

一旦您安装并正确配置了本插件，您在安装之后发布的文章的图片等文件将自动上传到阿里云OSS您的Bucket（包含各种size的图片和缩略图）。

== Installation ==

1. 下载，解压缩并上传到WordPress插件目录
2. 在插件管理后台激活插件
3. 工具 > OSS Support 

1. Download, unzip and upload to your WordPress plugins directory
2. activate the plugin within you WordPress Administration Backend
3. Go to Tools > OSS Support

== Frequently Asked Questions ==

Q：阿里云OSS免费么？如果不免费是怎么收费的？ A：阿里云OSS是收费服务，但有免费配额，详情请见http://www.aliyun.com/product/oss#price

Q：这是阿里云官方的插件么？ A：不是的。我只是一位PHPer，喜欢阿里云OSS服务，于是做了这个插件。

Q：本插件免费么？如果我不会用怎么办？ A：本插件基于GPL2.0协议开源发布，完全免费的。如果您使用过程中遇到任何问题请去 http://mawenjian.net/p/977.html 留言，我会尽我所能给您一个满意的答复。

Q：不是已经有一款类似的插件了吗，为什么要做重复工作？ A：因为那款插件只支持固定的上传目录'wp-content/uploads'，而我的博客因为历史原因，附件都放在了根目录，以致不能使用，所以我就在我的百度BCS插件的基础上修改，产生了此款插件。
